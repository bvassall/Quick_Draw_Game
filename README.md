# Quick Draw — Dead or Alive 🤠

A Wild West quick-draw shooting game built as a Progressive Web App (PWA). Test your reflexes in this addictive stick-figure showdown!

## Play

**[Play Online →](https://YOUR_USERNAME.github.io/quick-draw/)**

Or install as an app on your phone — just visit the link and tap "Add to Home Screen."

## Features

- 🔫 Fast-paced reflex-based gameplay
- 🏆 Leaderboard, achievements, and persistent stats
- 🏪 In-game shop with upgrades (Dual Wield, Bullet Time, and more)
- 💀 Multiple enemy types: Slow, Fast, Elite, and Boss
- 🎯 Headshot detection and speed ratings
- 📱 Fully offline — works as an installed PWA
- 🌵 Wild West aesthetic with procedural audio

## Hosting on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch**
4. Select the **main** branch and **/ (root)** folder
5. Click Save — your game will be live at `https://YOUR_USERNAME.github.io/REPO_NAME/`

## How to Play

- **Tap/click** enemies to shoot them before they shoot you
- **Headshots** (upper third of enemy) deal 3× damage
- Build **streaks** for score multipliers
- Survive rounds and spend gold in the **shop** for upgrades
- Compete against your own high scores on the **leaderboard**
